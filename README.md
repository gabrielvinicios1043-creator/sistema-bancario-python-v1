  Sistema Bancário em Python

Este projeto é um sistema bancário simples desenvolvido em Python com o objetivo de praticar lógica de programação e conceitos fundamentais da linguagem.

---

  Funcionalidades

-  Login com usuário e senha
-  Limite de tentativas de acesso
-  Visualização de saldo
-  Depósito de valores
-  Saque com verificação de saldo
-  Histórico de transações
-  Menu interativo

---

 Conceitos utilizados

- Estruturas condicionais (`if`, `elif`, `else`)
- Laços de repetição (`while`)
- Controle de fluxo (`break`, `continue`)
- Manipulação de listas (`append`)
- Entrada e saída de dados (`input`, `print`)

---

 Objetivo

Este projeto foi desenvolvido com foco em aprendizado prático, simulando um sistema bancário simples para reforçar conceitos de programação.

---

 Possíveis melhorias

- Tratamento de erros com `try/except`
- Organização do código em funções
- Interface mais amigável
- Persistência de dados (salvar informações)
